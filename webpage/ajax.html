Flickr API request

APIs (application programming interfaces) allow developers to send requests to third-party servers to get information and post data. JavaScript can execute these requests while the user is browsing and return the results to the page.

This brings up the issue/technique of AJAX (asynchronous javascript & xhtml). Ajax is when the javascript hits a code block to execute a request, instead of waiting for the server to respond before it moves on, javascript goes past and handles the rest of the code. The request response is handled in a CALLBACK function, which is always carried out after the response is returned.
